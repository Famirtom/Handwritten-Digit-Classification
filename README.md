# Handwritten-Digit-Classification
University machine learning project focused on handwritten digit classification using basic algorithms such as KNN and Multi-Layer Perceptron.


## Description
The goal of this project is to build and evaluate machine learning models that classify handwritten digits from a structured dataset. The main focus is on algorithm implementation, data preprocessing, model comparison, and performance evaluation.

## Algorithms Used
-**k-Nearest Neighbours (KNN)**
-**Multi/Layer Perceptron (MLP)**
-**Genetic Algorithm (GA)** for model optimisation and parameter tuning.

## Key Features
- Data preprocessing and feature handling
- Implementation of basic machine learning models
- Use of Genetic Algorithms for optimisation
- Model evaluation using accuracy and confusion matrices

## Tools
Java
CSV datasets

## Learning Outcomes
- Understanding basic machine learnign models
- Preprocessing and exploring data
- Analysing model performance
