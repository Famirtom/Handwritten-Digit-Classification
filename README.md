# Handwritten Digit Classification

## Overview
This project implements a handwritten digit classification system built from scratch in Java. The goal is to classify digit images using machine learning techniques without relying on high-level frameworks.

## Features
- k-Nearest Neighbour (k-NN) implementation
- Multi-Layer Perceptron (MLP) neural network
- Genetic Algorithm for hyperparameter optimisation
- Model evaluation using confusion matrix and accuracy metrics

## Technologies
- Java
- Custom ML implementation (no external ML libraries)

## Results
- Achieved ~98% classification accuracy
- Compared performance between k-NN and MLP models

## Key Learning
This project demonstrates understanding of:
- Machine learning fundamentals
- Model evaluation techniques
- Algorithm design and optimisation

## How to Run
1. Clone the repository
2. Compile the Java files
3. Run the main class
